# D2R_Mod_jcy 暗黑破坏神II：狱火重生 Mod

## 简介
* 本 MOD 基于 Will_v2.8 制作，整合了各种增强包（加油包），通过控制器可以自由调整配置，满足不同需求。

* 仅面向 Windows PC 用户，不支持手柄操作。

* 运行控制器程序：
  Mods/jcy/jcy 控制器.exe
  修改设置后点击 “应用设置” 保存配置，重启游戏后生效。
  
* 如果有疑虑，可以直接删除控制器程序，这不会影响 MOD 的默认功能；
  或者可以从代码仓库获取源码，自行打包使用。
  仓库地址：https://github.com/jcymeow/jcy

## 使用
* 请先下载页面中的全部附件，
  然后解压其中任意一个压缩包即可（内容是一样的）。

* 当需要更新 MOD 时，
  请先删除旧版本的 MOD 目录，
  再解压并使用最新版本，避免残留文件导致异常。

* 游戏设置, 勾选"额外的命令行参数"，需要在启动参数中加入以下内容：
  `-mod jcy -txt -w -assettestmode 1`
  其中：
  -mod jcy -txt 是 JCY MOD 必须的基础参数
  -w 表示 以窗口模式运行游戏（不需要可以删掉）
  -assettestmode 1 是 国服专用参数，用于将怪物死亡后的墓碑恢复为尸体
  请根据你自己的客户端版本和使用需求选择是否启用这些参数。
  每个参数之间必须用空格分隔，否则游戏无法正确识别。

* 控制器程序和 MOD 包是绑定在一起使用的，不能分开存放。
  请务必将控制器程序放在 MOD 目录内，并从以下位置运行：
  Diablo II Resurrected/Mods/jcy/jcy 控制器.exe
  如果在其他位置运行，程序将无法正确识别 MOD 文件。
  
* 如果你电脑里装了多个《暗黑破坏神 II：重制版》客户端，只需要在其中一个客户端里更新和使用 MOD 就够了。
  其他客户端可以通过 目录链接 的方式，直接共享这个 MOD 目录，不用再复制一份。
  下面这条命令的意思是：让客户端 2 的 Mods/jcy 指向客户端 1 的 Mods/jcy
  `mklink /D "客户端2\Diablo II Resurrected\Mods\jcy" "客户端1\Diablo II Resurrected\Mods\jcy"`
  其中:  
  /D 创建目录链接（固定写法）
  前面的是「要生成的目录」
  后面的是「真正存放文件的目录」
  需要 以管理员身份运行 CMD

* Handle -- 杀进程用
   Handle 是一个实用工具，用于显示有关系统中任何进程的打开句柄的信息。 可以使用它查看打开了文件的程序，或查看程序的所有句柄的对象类型和名称。
